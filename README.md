# A M Pabel | Portfolio 🚀

Welcome to my official portfolio repository. I am a **Motion Artist**, **Video Editor**, and **Graphic Designer** dedicated to crafting visual stories that resonate.

## 🌐 Live Demo
View the live site here: **https://pabeldev.github.io/A-M-Pabel/**

## 🧑‍💻 About Me
My creative journey began in **2019** as a hobbyist. Over the years, I transformed that passion into a professional career.
* **Professional Graphic Design:** Active since 2022.
* **Motion Graphics & Video Editing:** My primary focus since 2024.
* **The Goal:** To breathe life into static designs and tell compelling brand stories through motion.

## 🛠️ Technical Stack
This portfolio is built using:
* **Frontend:** HTML5, CSS3 (Custom Animations & Flexbox/Grid)
* **Typography:** Google Fonts (Poppins & Hind Siliguri)
* **Icons:** Font Awesome 6.5.1
* **Forms:** Integrated with Formspree for seamless communication

## 🌟 Key Features
- **Dynamic Work Gallery:** Toggle between Video/Motion and Graphic Design projects.
- **Interactive Media:** One-click YouTube video playback directly within the site.
- **Brand Showcase:** Highlighting collaborations with brands like Infinity Mega Mall and Canada Shikor TV.
- **Responsive Design:** Optimized for Desktop, Tablet, and Mobile devices.

## 📈 Roadmap
I am currently evolving this project from a static HTML site to a **Next.js** framework to implement better performance and modern web standards.
